# Vulcanus ansible setup
Based on [Vulcanus] (https://github.com/FoioK/Vulcanus)

## Features:
* Automatic Let's Encrypt certificate
* Load balanced via nginx (with stats gathering from all the nodes)
* Grafana support (With influxDB)
